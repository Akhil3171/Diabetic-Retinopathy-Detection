# Diabetic-Retinopathy-Detection
In this project, we try to evaluate Diabetic retinopathy eye condition. We have different classes based on the severity of the condition.

0 - No DR
1 - Mild
2 - Moderate
3 - Severe
4 - Proliferative DR
